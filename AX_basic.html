02. AI 모델 & AI 에이전트란 무엇인가?

Copilot을 효과적으로 활용하려면, 그 뒤에서 작동하는 AI의 기본 개념을 이해하는 것이 중요합니다.
복잡한 수식이나 코드를 몰라도 됩니다. AI가 어떻게 생각하고 행동하는지에 대한 직관적인 이해만으로 충분합니다.

1 AI 모델이란 무엇인가?

한 줄 정의

AI 모델은 방대한 텍스트 데이터를 학습하여 언어를 이해하고 생성할 수 있도록 훈련된 수학적 시스템입니다.

사람으로 비유하자면, 수천만 권의 책을 읽고 그 지식을 기반으로 대화하는 초고속 두뇌와 같습니다.

📌 대표적인 AI 모델들

GPT-4oOpenAI가 개발. ChatGPT의 기반 모델. 빠른 응답과 높은 범용성.
ClaudeAnthropic이 개발. 긴 문서 처리·자연스러운 글쓰기에 특화.
GeminiGoogle이 개발. 검색·이미지 분석 등 멀티모달 강점.

ChatGPT vs Copilot — 핵심 차이

ChatGPT는 OpenAI가 자체 개발한 GPT 모델만을 사용하는 서비스입니다.

반면 Microsoft Copilot은 특정 AI 모델에 종속되지 않습니다. Copilot은 GPT, Claude 등 다양한 최첨단 AI 모델을 상황에 맞게 선택하여 활용할 수 있는 플랫폼입니다.

💡 쉬운 비유

ChatGPT = 특정 브랜드의 커피만 파는 카페
Copilot = 다양한 브랜드의 최고급 원두를 선택해서 내려주는 카페

아웃룩 Copilot에서 Claude 선택 가능!
현재 아웃룩 코파일럿에서는 좌측 상단 '자동' 버튼을 통해 GPT 또는 Claude 모델을 직접 선택할 수 있습니다.

2 AI 에이전트(Agent)란 무엇인가?

AI 에이전트 = 스스로 판단하고 행동하는 AI

기존 AI 모델은 질문에 답변만 했습니다. AI 에이전트는 한 걸음 더 나아가, 목표를 주면 스스로 계획을 세우고, 여러 도구를 사용하여, 단계적으로 실행까지 합니다.

쉽게 말해, AI 모델이 "두뇌"라면, AI 에이전트는 그 두뇌를 가지고 직접 일을 처리하는 직원입니다.

일반 AI (채팅)

"이메일 초안 작성해줘"
→ 텍스트 답변 제공 후 종료

단순 질의응답
진화

AI 에이전트

"이번 주 이메일 정리해서
주간보고 초안 만들어줘"
→ 메일 검색 → 분류 → 작성

계획 → 실행 → 완료

멀티 에이전트

여러 AI 에이전트가
협업하여 복잡한 업무를
분업·처리

미래의 업무 방식

정리: Copilot은 어디에 해당할까요?

Microsoft Copilot은 현재 AI 모델 + 에이전트 기능을 모두 갖춘 서비스입니다.

채팅을 통해 답변을 받는 것은 AI 모델 활용이고,
Copilot이 이메일을 자동으로 정리하거나 파일을 검색하는 것은 에이전트 기능입니다.

🔑 오늘 교육의 핵심 관점

지금 우리가 배우는 "복사 → 붙여넣기 → 전송" 방식은 AI 모델을 활용하는 가장 기초적이고 강력한 방법입니다. 이 기초 위에 에이전트 자동화가 쌓입니다.

베이스 모델: GPT / Claude 선택 가능

자체 모델이 아닌 최고의 외부 모델을 연결하여 활용

사내 보안 적용된 안전한 환경

M365 계정 로그인 시 입력 데이터가 AI 학습에 활용되지 않음

Office 앱과 자연스럽게 연동

Outlook, Excel, Teams 등 업무 도구와 통합

03. Copilot, 어떻게 쓰는 건가요?

"Copilot = 회사 계정으로 로그인한 AI 채팅 서비스"

ChatGPT처럼 말을 걸면 대답하는데, 사내 보안이 적용된 Microsoft 공식 서비스입니다.
별도 설치 없이 브라우저에서 바로 씁니다. → m365.cloud.microsoft/chat

🚀 처음이라면 이 3단계만 기억하세요

1

복사 (Ctrl+C)

이메일 본문, 엑셀 표, PDF 내용 등
필요한 텍스트를 복사합니다

2

붙여넣기 + 지시

Copilot 입력창에 붙여넣고
"이걸 한글로 요약해줘"처럼 말합니다

3

결과 확인 후 활용

AI 답변을 그대로 쓰지 말고
반드시 내용을 검토 후 수정하세요

💡 이것만 바꿔도 답변 품질이 달라집니다

이렇게 물으면 → 두루뭉술한 답변

"이메일 써줘"
"이 데이터 분석해줘"

누구에게, 어떤 목적으로, 어떤 어조로 쓸지 정보가 없어 엉뚱한 결과가 나옵니다.

이렇게 물으면 → 바로 쓸 수 있는 답변

"브라질 법인 담당자에게 수리 지연 사과와 보증 연장 제안을 담은 영문 메일 초안 써줘"
"아래 CSAT 데이터에서 점수가 가장 낮은 카테고리 3개를 표로 정리해줘 [데이터 붙여넣기]"

목적 + 조건 + 데이터를 함께 넣으면 바로 사용 가능한 결과가 나옵니다.

처음 쓰는 분들이 자주 묻는 질문

Q.
Copilot에 입력한 내용이 외부로 새나가지 않나요?
사내 M365 계정으로 로그인한 상태에서는 입력 내용이 AI 학습에 활용되지 않습니다. 단, 미인증 개인 계정 사용 시에는 주의가 필요합니다.
Q.
AI가 틀린 정보를 줄 수도 있나요?
네, 있습니다. Copilot은 확률적으로 답하기 때문에 수치·날짜·고유명사는 반드시 직접 검토하고 사용하세요.
Q.
원하는 답이 안 나오면 어떻게 하나요?
포기하지 말고 "더 짧게 줄여줘", "불렛포인트로 바꿔줘", "이 부분을 구체적으로 써줘"처럼 추가 요청(Follow-up)을 해보세요.

04. 브라우저 기반 Copilot 핵심 기능 도감

아래 정리된 기능들은 MS Edge 브라우저와 웹 페이지만 열려 있다면 별도의 복잡한 절차 없이 상시 사용 가능한 실무 핵심 기능들입니다.

Edge 사이드바 문서 요약

Edge 브라우저 오른쪽 화면에 Copilot을 띄워 현재 탭을 분석하는 기능

🔰 이런 분께 추천: 영문 보고서나 긴 PDF를 빠르게 파악해야 할 때
  • PDF/Word/웹 페이지 분석: 마케팅 트렌드 보고서, 뉴스, 사내 웹 문서를 스크롤 내리지 않고 즉시 요약.
  • 영문 문서 즉시 번역: 해외 시장 조사 보고서 등의 핵심 지표만 골라내 한글 보고서용으로 재가공.

Copilot Web 초안 빌더

m365.cloud.microsoft/chat 웹 포털을 켜서 복사해 온 데이터로 텍스트를 창작하는 기능

🔰 이런 분께 추천: 영문 메일이나 보고서 초안 작성에 시간이 많이 걸릴 때
  • 비즈니스 메일 초안: "단가 15% 인하 거절, 대신 보증기간 1년 연장" 등 키워드만 던져도 공손한 영문/다국어 이메일 생성.
  • 보고서 뼈대 구성: 아이디어나 개요 메모를 붙여넣고 공식 보고서 톤으로 변환 및 목차 구성.

엑셀 수식 & 데이터 도우미

엑셀 파일 내 수식 설계 및 복사해 온 데이터 테이블 가공

🔰 이런 분께 추천: 엑셀 수식이 헷갈리거나, 데이터를 빨리 표로 정리해야 할 때
  • 대화형 수식 제안: 원하는 동작을 한글로 입력하면 XLOOKUP, INDEX-MATCH 등 난이도 높은 수식 제작 및 작동법 설명.
  • 텍스트 테이블 분석: 엑셀 표 영역을 긁어서 입력창에 붙여넣으면 평균, 이상치, 단순 트렌드 도출.

05. 업무 효율을 극대화하는 2대 접속 공식

사내 PC에서 가장 신속하고 범용적으로 Copilot을 호출하는 두 가지 대표 경로입니다. 이 두 경로를 통해 오피스 문서 및 웹 정보를 끊김 없이 연동할 수 있습니다.

처음 접속 시 로그인 확인: 접속 후 오른쪽 상단 프로필 아이콘을 클릭하여 사내 계정(@lge.com)으로 로그인되어 있는지 반드시 확인하세요. 개인 계정으로 로그인된 경우 보안 이슈가 발생할 수 있습니다.

브라우저 연동

공식 1: MS Edge 브라우저 사이드바

웹서핑이나 사내 그룹웨어(메일, 결재), PDF를 읽으며 실시간으로 조작할 때 가장 강력합니다.

[실행법]
  1. MS Edge 브라우저를 실행합니다.
  2. 브라우저 오른쪽 최상단 구석에 위치한 파란색 Copilot 아이콘을 마우스로 클릭합니다.
  3. 또는 키보드 단축키 Ctrl + Shift + . (마침표)를 동시에 누릅니다. ※ Edge 버전에 따라 단축키가 다를 수 있습니다. 아이콘 직접 클릭을 권장합니다.
독립형 웹 포털

공식 2: Copilot 전용 웹 포털 접속

대량의 텍스트 복사-붙여넣기, 수식 작성을 요청할 때 넓고 쾌적한 화면에서 실행 가능합니다.

[실행법]
  1. 크롬(Chrome)이나 에지 브라우저에서 아래 전용 웹 주소에 접속합니다.
  2. m365.cloud.microsoft/chat
  3. 사내 계정(또는 개인 Microsoft 계정)으로 로그인하면, 상용 수준의 AI 엔진을 넓은 화면에서 조작할 수 있습니다.

06. [실전 A to Z] PDF 및 웹 문서 요약하기

실무 상황

CC&S전략팀이 공유한 'lg_electronics_cc&s_transformation.pdf' (영문 38페이지) 파일이 팀즈에 업로드되었습니다. LG전자 수리/상담 서비스 개선 방향 수립을 위해 핵심 인사이트를 파악해야 하지만, 전문 영문 보고서를 전부 읽을 여유가 없습니다.

1

PDF 문서 열기

바탕화면에 있는 PDF 파일을 마우스로 클릭하여 MS Edge 브라우저 화면 안으로 끌어다 놓습니다. (Edge에서 PDF 뷰어가 실행됩니다.)

2

Edge Copilot 사이드바 호출

Edge 브라우저 우측 상단의 파란색 Copilot 아이콘을 클릭하거나, 단축키 Ctrl + Shift + .을 눌러 우측 채팅창을 활성화합니다.

07. [실전 A to Z] Outlook 이메일 연동 작성법

실무 상황

해외법인 CS 담당자(Maria Santos, 브라질 법인)로부터 특정 제품 수리 처리 지연에 대한 강한 항의 이메일이 접수되었습니다. 해당 건은 부품 수급 지연이 원인이며, 무상 연장 보증(+6개월) 및 우선 처리 약속을 제안하며 정중하게 상황을 해명하는 영문 회신을 작성해야 합니다.

꼭 알아야 할 아웃룩 Copilot 핵심 규칙

①

아웃룩 이메일 데이터는 '아웃룩 코파일럿'에서만 연동됩니다.
아웃룩 앱이 열려 있을 때 아웃룩 화면 좌측 사이드바에 있는 Copilot 버튼을 눌러야만 메일함 데이터가 AI와 연동됩니다. Copilot Web(m365.cloud.microsoft)이나 Edge 사이드바 코파일럿에서는 아웃룩 메일 데이터에 직접 접근이 불가합니다.

②

무료 버전 사용자는 아웃룩 코파일럿만 사용 가능합니다.
M365 무료(기본) 계정 사용자는 Copilot Web 등 다른 코파일럿에서 아웃룩 이메일 데이터가 연동되지 않습니다. 따라서 아웃룩 앱 내 사이드바 Copilot 버튼이 유일한 이메일 AI 연동 경로입니다.

✦

🔥 아웃룩 Copilot에서만 가능한 숨겨진 기능: Claude 모델 선택!
현재 아웃룩 Copilot에서는 다른 코파일럿과 달리 AI 모델을 Claude(클로드)로 변경할 수 있습니다. 아웃룩 Copilot을 열면 좌측 상단에 '자동'이라고 표시된 버튼이 있는데, 이 버튼을 눌러 Claude로 모델을 변경하면 더 자연스럽고 풍부한 이메일 답변 초안을 얻을 수 있습니다.

복사해서 바로 쓰는 프롬프트
아래 해외법인 담당자의 항의 이메일에 대해, 다음 조건을 바탕으로 격식 있고 정중한 비즈니스 영문 답장 이메일(English Business Email) 초안을 작성해 줘. [해명 사유 및 제안 내용] 1. 해당 수리 건 처리가 지연된 이유는 특정 부품(part)의 글로벌 수급 지연 때문이며, 당사의 귀책이 아닌 공급망 이슈임을 정중하게 설명. 2. 고객 불편에 대한 진심 어린 사과와 함께, 무상 보증 기간을 6개월 추가 연장하는 보상안 제시. 3. 해당 수리 건을 최우선(Priority) 처리 대상으로 지정하여 빠른 시일 내 완료하겠다는 약속 포함. 4. 추가 문의 사항은 직접 연락 요망 (담당자 직통 연락처 포함 안내). [어조 및 포맷] - 격식 있고 전문적인(Formal, Professional) 영미권 비즈니스 레터 스타일. - 알맞은 인사말(Dear Maria,)과 맺음말(Best regards,)을 포함할 것. [Maria Santos 수신 메일 본문] (여기에 복사해 둔 이메일 내용을 붙여넣으세요)

08. [실무 A to Z] 주간보고 요약자료 만들기

실무 상황

매주 각 팀에서 취합된 진척 현황 내용을 Copilot에 붙여넣으면, CC&S전략팀 W주차 주간업무 보고 형식의 깔끔한 A4 보고 양식으로 자동 변환해 줍니다. 팀별 내용을 2~3줄로 핵심만 압축하여 경영진 보고용 요약자료를 빠르게 완성하세요.

주간보고 요약 시 추천: 아웃룩 Copilot + Claude 모델 조합

주간보고 이메일이 아웃룩에 모여 있다면, 아웃룩 Copilot을 활용하는 것이 가장 효과적입니다. 아웃룩 코파일럿은 메일함 데이터를 직접 읽어 요약할 수 있으며, 특히 Claude 모델로 변경 시 더 자연스럽고 정교한 요약이 가능합니다.

아웃룩 Copilot + Claude 실행 순서

Step 1

아웃룩 앱을 열고 좌측 사이드바의 Copilot 버튼을 클릭하여 아웃룩 Copilot을 실행합니다.

Step 2

코파일럿 창 좌측 상단의 '자동' 버튼을 클릭합니다.

Step 3

모델 선택 목록에서 Claude(클로드)를 선택하여 변경합니다.

Step 4

아래 프롬프트를 입력하거나, 주간보고 관련 이메일을 열고 내용 요약을 요청합니다.

참고: 아웃룩 Copilot의 Claude 모델 선택은 현재 아웃룩 코파일럿에서만 가능한 기능입니다. 다른 코파일럿(Copilot Web, Edge 사이드바 등)에서는 지원되지 않을 수 있습니다.
