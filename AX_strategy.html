AI 운영 인프라 플랫폼(샌드박스) 활용
💡 왜 필요한가요?
지금은 AI 도구를 내 PC에서만 실행하다 보니, PC를 끄면 멈추고 다른 사람과 공유도 어렵습니다.
이를 해결하기 위해 회사 서버에 나만의 AI 작업 공간을 만들어 드립니다.
내 PC가 꺼져 있어도 AI가 계속 돌아가고, 만든 결과물을 팀원들과 바로 공유할 수 있습니다.

4. [문화] AX가 지속되는 조직 문화 만들기

※ 인프라 · 역량 · 실행이 AX를 가능하게 만든다면, 문화는 AX가 사라지지 않고 지속 · 확산되게 만드는 토양입니다.

안정감
"실패해도 괜찮다"는 분위기가 AX 시도의 문을 엽니다.
실패 사례 공유 게시판 운영
커뮤니티 내 실패 경험 공유 공간 개설 ➜ 비효율 프롬프트 · 잘못된 결과물도 솔직하게 게시하여 시도 자체를 자산화
AX Stars 수상 외 과제 피드백
수상하지 않은 제출 과제에도 긍정 피드백 및 발전 가능성 공식 검토 ➜ 참여 허들을 낮추고 재도전 동기 유지
리더의 방향성 정립 및 지시
리더가 업무 적용 방향성을 명확히 지시하고 드라이브합니다.
업무 선반영 지시 및 방향성 정립
실무 업무에 AI를 선반영하여 일할 수 있도록 적극적으로 지시하고, 부문 AX 추진 방향성을 명확히 정립 ➜ 강력한 실행력 확보
과제 인센티브에 AX 평가 지표 반영
AI 활용도 · 기여도를 과제 인센티브 평가 항목에 공식 포함 ➜ 리더가 직접 기준을 설계하고 체험하는 자기강화 구조 구축
AX, 일의 방식으로 체화
일상 루틴으로 AX가 업무 기본값이 되어야 합니다.
기존 업무 루틴에 AX 삽입
내가 하고 있는 일들을 AI에게 알려주고, 실행 가능 여부를 직접 확인하는 것부터 시작
  • 주간 보고를 AI로 초안 생성하여 보고
  • 중점과제 발표자료 전체 입력 ➜ AI 평가점수 산출
  • 회의록 자동 작성 및 액션 아이템 정리
핵심 메시지
문화는 인프라 · 역량 · 실행 3축이 자기강화 선순환으로 돌아가게 하는 접착제입니다. 안정감이 시도를 만들고 ➜ 리더 실천이 확산을 견인하고 ➜ 일상 루틴 체화가 AX를 프로젝트가 아닌 일하는 방식으로 전환합니다.

EAP 샌드박스 플랫폼 소개

EAP 샌드박스란?
사내 데이터 분석가와 실무자들이 보안 정책을 준수하면서 안전하게 대용량 데이터를 수집/분석하고,AI를 개발·검증할 수 있도록 전사 차원에서 제공하는 클라우드 기반 가상 컴퓨터 환경입니다.
주요 제공 기능 및 혜택
• 분석 및 실습 전용 공간: 개인 PC 성능이나 회사 보안 정책에 지장을 주지 않는 완전히 분리된 독립 개발 영역
• 개발 인프라 기본 탑재: Python 언어, Jupyter Lab, VS Code Web 등 데이터 분석에 최적화된 최신 도구 제공
• 데이터 직접 연동: CS 데이터마트를 포함한 내부 정형/비정형 원천 데이터를 안전하게 연결하여 실시간 처리 가능
운영 주체 및 접속 경로
운영 주체: AX업무혁신담당 데이터플랫폼실
접속 경로: EAP 데이터분석 Sandbox MLDL
💡 참고 (Sandbox 개념):
어린아이들이 노는 모래통(Sandbox)에서 유래하여, 외부 실제 환경이나 운영 중인 시스템에 나쁜 영향을 주지 않고 안전하게 여러 기능을 만들어보고 테스트해 볼 수 있도록 격리된 공간을 의미합니다.
